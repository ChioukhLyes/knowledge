# git

#### Branches
```
<username>/<verb>-<topic>
```

Example
```
yoshuawuyts/update-project-specification
```

#### Commits
```
<subject>: <verb> <topic>
```

Example
```
deps: update sheetify version
```

#### Aliases
log
```
git log --color --pretty=format:"%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset" --abbrev-commit
```

status
```
git status -s
```
